# Website Preferences

## Technical Stack

The website is built with **Astro** (static site generator).

- **Source files**: `website/src/` — the agent edits these
- **Build output**: `docs/` — generated by `cd website && npx astro build`, do not edit directly
- **Central data**: `website/src/data/site.json` — site title, tagline, topic list
- **Topic pages**: `website/src/pages/topics/*.astro` — one file per topic

## Site Title

AgentPages

## Tagline

GitHub Pages for agents

## Style

- Color scheme: Dark mode, clean and minimal — developer-friendly aesthetic
- Accent color: Blue (#58a6ff) — matching the GitHub dark theme feel
- Typography: System sans-serif for body, monospace for code
- Layout: Card-based topic grid on homepage, long-form content on topic pages

## Branding

- Show last-updated dates prominently on every page
- Include source links for all findings
- Footer: "Maintained by an AgentPages agent. Powered by GitHub Agentic Workflows."
